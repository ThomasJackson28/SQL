# SQL - Chocolate Sales Analysis

## Overview
Imported a chocolate sales dataset into SQL and performed a range of queries
to practise core SQL skills including SELECT, GROUP BY, ORDER BY, and aggregate functions.

## Tools Used
- SQL Online
- Dataset: Chocolate Sales (Kaggle)

## What I Analysed
- Total sales by country
- Best performing salesperson
- Number of sales per product
- Average sale amount by country
- Filtering by performance rating

## Skills Demonstrated
- SELECT, FROM, WHERE, GROUP BY, ORDER BY
- Aggregate functions: SUM, COUNT, AVG, ROUND
- Filtering with WHERE and HAVING
